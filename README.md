# Inverse-Kinematics-for-Human-Fingers
A research project that I did for my BSc level Robotics course

The primary objective of this research project will be to study the kinematics of the human
fingers. The aim is to put the fingertip on a specific target location or move along a pre-determined trajectory. To do this, the inverse kinematics of the model needs to be determined. Usually, the best method to do this is to look at the geometrical relations within the model to find an analytical solution. However, if not enough relations are given, an analytical solution cannot be found and numerical or iterative methods need to be used instead. As it turns out, these methods can sometimes give problems or a non-sufficient solution. It will be the goal of this project to optimize an iterative method as far as possible, such that a sufficient solution will be returned no matter the desired target positions (provided that these positions are within the reach of the finger).
